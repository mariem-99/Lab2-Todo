# ✨ Todo App — Lab 2 Challenge

A beautiful, interactive Todo application built with React as part of Lab 2: Components, Props & State.

## 🚀 Live Demo
[View on Vercel](https://todo-app-three-weld-82.vercel.app)

## 📸 Preview
![Preview](public\todo.png)

## 🛠 Built With
- React
- useState Hook
- Plain CSS (Glassmorphism design)

## ✨ Features
- ✅ Add new todos
- ✅ Mark todos as complete
- ✅ Delete todos
- ✅ Filter by All / Active / Completed
- ✅ Live stats (Total, Done, Remaining)
- ✅ Beautiful dark pink glassmorphism UI

## 📦 How to Run Locally
```bash
git clone https://github.com/mariem-99/Lab2-Todo.git
cd lab-2-todo
npm install
npm start
```

## 🧠 Key Concepts Practiced
- `useState` for managing arrays and state
- Lifting state up to parent component
- Controlled components
- Component communication via props and callbacks
- Conditional rendering

## 📁 Project Structure
```
src/
├── components/
│   ├── TodoForm.js      # Input form to add todos
│   ├── TodoList.js      # Renders list of todos
│   ├── TodoItem.js      # Individual todo item
│   ├── TodoStats.js     # Shows total/done/remaining
│   └── TodoFilter.js    # All/Active/Completed filter
├── App.js               # Main app, manages all state
└── App.css              # Glassmorphism styling
```
